# [ECode](https://pengfeiw.github.io/ecode/)

A online code editor for web develop. Try it [here](https://pengfeiw.github.io/ecode/).

## Run in local

- `yarn install`
- `yarn start`
- open `[localhost:9000](http://localhost:9000)`
